# Python学练写一体化网站 - 产品需求文档

## 概述
- **摘要**：开发一款Python学练写一体化网站，涵盖课程学习、项目训练、代码实操场和视频教程，提供沉浸式学习体验，包括代码编辑运行、互动学习、用户系统、个性化推荐和社区功能。
- **目的**：为学习者提供一个全面的Python学习平台，通过实践和互动方式提升学习效果，构建学习社区促进知识共享。
- **目标用户**：Python初学者、中级开发者、希望提升Python技能的专业人士。

## 目标
- 提供完整的Python学习课程体系，从入门到进阶
- 实现代码在线编辑和运行功能，支持实时反馈
- 开发互动式学习模块，增强学习体验
- 设计10个实战项目，巩固学习成果
- 提供视频教程，辅助课程学习
- 建立用户注册登录系统，记录学习进度
- 实现个性化学习路径推荐功能
- 构建社区交流平台和成就激励系统

## 非目标（范围外）
- 不提供离线学习功能
- 不支持其他编程语言
- 不包含企业级培训内容
- 不提供证书认证服务
- 不开发移动应用

## 背景与上下文
- Python作为一种广泛使用的编程语言，需求持续增长
- 传统学习方式缺乏互动性和实践机会
- 在线教育平台需要提供更沉浸式的学习体验
- 社区互动和个性化推荐能显著提升学习效果
- 视频教程已成为现代教育的重要组成部分

## 功能需求
- **FR-1**：课程系统 - 从入门到进阶的课程体系，包含10-15个章节，每节课包含文字讲解、代码示例、习题
- **FR-2**：项目训练 - 10个实战项目，覆盖常用场景，每个项目提供任务描述、参考代码、扩展挑战
- **FR-3**：代码实操场 - 在线编写Python代码，支持代码高亮、自动补全，左侧代码编辑器，右侧实时显示示例代码或运行结果，支持运行代码并返回结果（安全沙箱）
- **FR-4**：视频教程 - 每个章节配套5-10分钟短视频，可上传或嵌入第三方视频链接
- **FR-5**：用户系统 - 支持注册、登录、个人资料管理、进度追踪、收藏
- **FR-6**：学习进度跟踪 - 记录用户学习历史和进度
- **FR-7**：习题验证 - 自动判断代码正确性
- **FR-8**：社区讨论区 - 支持问答、项目分享
- **FR-9**：个性化学习路径 - 根据用户水平和兴趣推荐学习内容
- **FR-10**：成就系统 - 设置学习目标和奖励机制，激励用户学习
- **FR-11**：主题切换 - 支持暗色/亮色主题切换

## 非功能需求
- **NFR-1**：性能 - 代码运行响应时间不超过3秒
- **NFR-2**：可用性 - 平台界面友好，操作简单直观
- **NFR-3**：安全性 - 保护用户数据和代码安全，安全执行用户提交的Python代码
- **NFR-4**：可扩展性 - 支持平台功能的持续更新和扩展
- **NFR-5**：兼容性 - 支持主流浏览器访问，响应式设计适配不同设备
- **NFR-6**：用户体验 - 界面美观，交互流畅，加载速度快

## 约束
- **技术**：使用React + TypeScript + Express.js构建全栈应用
- **依赖**：使用Supabase进行用户认证和数据存储
- **安全**：使用Docker沙箱或Pyodide安全执行用户代码
- **预算**：开发成本控制在合理范围内
- **时间**：项目开发周期为3个月，分阶段开发

## 假设
- 用户具备基本的计算机操作能力
- 网络环境稳定，能够流畅访问在线平台
- 用户对Python学习有明确的需求和兴趣
- 服务器具备足够的资源运行代码沙箱

## 验收标准

### AC-1：课程系统
- **给定**：用户进入课程页面
- **当**：用户浏览课程列表并选择章节
- **则**：系统展示课程内容，包括文字讲解、代码示例和习题
- **验证**：`human-judgment`

### AC-2：项目训练
- **给定**：用户进入项目页面
- **当**：用户选择并开始一个项目
- **则**：系统提供项目描述、要求、参考代码和扩展挑战
- **验证**：`human-judgment`

### AC-3：代码实操场
- **给定**：用户进入代码实操场
- **当**：用户编写Python代码并点击运行按钮
- **则**：系统在安全沙箱中执行代码并显示运行结果
- **验证**：`programmatic`

### AC-4：视频教程
- **给定**：用户进入课程章节页面
- **当**：用户点击视频教程链接
- **则**：系统播放视频教程或跳转到第三方视频平台
- **验证**：`human-judgment`

### AC-5：用户注册登录
- **给定**：用户访问平台
- **当**：用户注册新账号或登录现有账号
- **则**：系统验证用户身份并创建或加载用户资料
- **验证**：`programmatic`

### AC-6：学习进度跟踪
- **给定**：用户登录平台
- **当**：用户学习课程或完成练习
- **则**：系统自动记录学习进度
- **验证**：`programmatic`

### AC-7：习题验证
- **给定**：用户完成习题
- **当**：用户提交代码
- **则**：系统自动判断代码正确性并提供反馈
- **验证**：`programmatic`

### AC-8：社区讨论区
- **给定**：用户进入社区页面
- **当**：用户发布帖子或回复他人
- **则**：系统展示社区内容并支持互动
- **验证**：`programmatic`

### AC-9：个性化学习路径
- **给定**：用户完成初始评估或有学习历史
- **当**：用户浏览课程推荐
- **则**：系统根据用户情况推荐个性化学习路径
- **验证**：`human-judgment`

### AC-10：成就系统
- **给定**：用户完成学习任务或达到学习目标
- **当**：系统检测到用户成就
- **则**：系统颁发成就徽章并通知用户
- **验证**：`programmatic`

### AC-11：主题切换
- **给定**：用户访问平台
- **当**：用户切换主题设置
- **则**：系统切换到暗色或亮色主题
- **验证**：`human-judgment`
